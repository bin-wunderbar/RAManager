Acerca de RA-Manager

(Versión 3.0, Java SE 1.8, Eclipse 2021-09 - Grupo 4 - CIFP Txurdinaga 1DW)

Uso

Aplicación para la gestión de los talleres Rekord Autoak S.L.
Gestión completa de procesos y entidades
Proceso de gestión de encargos por empleado
Impresión directa en impresoras del sistema
Registro de eventos
Multilenguage
Gestión de usuarios y contraseñas
Gestión de permisos y roles
Integración con el home de usuario
Recuperación ante fallos de desconexión
Registro de md5 de contraseña en base de datos Cifrado en disco de la contraseña del usuario de conexión de la base de datos

Tecnologías

Java SE, awt, swing, LGoodDatePicker
Diseño de capas
Persistencia en base de datos mariadb
